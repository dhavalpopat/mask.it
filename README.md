# mask.it
This is a computer vision project to apply various masks on facial images using landmark detection and mask morphing.

### Implementation
The project was divided into four modules:
<ol>
<li>Face detection</li>
<li>Facial landmark detection</li>
<li>Morphing the mask</li>
<li>Overlaying the mask to face</li>
</ol>
